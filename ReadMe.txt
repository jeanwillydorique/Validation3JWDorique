Bonjour voici les manipulations à faire.

Veuiller copier le script 1 et le script 2 dans le dossier ou vous souhaitez créer la Machine Virtuelle.

en CLI:
bash script1

et suivre les instructions.

Une fois la machine créer vous serez à l'interieur de celle-ci.
Bravo !!! 

les commandes à effectuer pour le second script 

Pour sortir de la VM :
exit 


déplacer le script 2 dans le dossier synchronisé crée par la machine.

retourné dans la VM :
vagrant ssh

puis vous rendre dans le dosser synchronisé :
cd /var/www/html 

et enfin lancer le script 2 :
bash script2

et voila votre VM est prête à être utilisée
